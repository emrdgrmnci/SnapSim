## Design Ideas

### Option 1: iPhone with Arrow
- Simple iPhone outline
- Down arrow indicating "snap to corner"
- Clean, minimalist design
- Colors: iOS blue gradient (#007AFF to #5AC8FA)

### Option 2: iPhone in Corner
- iPhone silhouette
- Positioned in corner with motion lines
- Represents the hiding action
- Colors: White iPhone on gradient background

### Option 3: Layered iPhones
- Two overlapping iPhone shapes
- One visible, one fading
- Represents show/hide
- Colors: Gradient from visible to transparent

## Specifications

- **Size:** 1024x1024 px (for macOS)
- **Format:** PNG with transparency
- **Style:** Flat design, modern
- **Background:** Gradient or solid color
- **Corners:** Rounded (macOS style)

## Tools to Create Icon

### Free Options:
1. **Figma** (https://figma.com) - Professional design tool
2. **Canva** (https://canva.com) - Easy templates
3. **SF Symbols** - Use Apple's built-in symbols

### AI Options:
1. **DALL-E** - Generate with prompt: "Modern macOS app icon, iPhone simulator snapping to corner, minimalist, gradient blue"
2. **Midjourney** - Similar prompt
3. **Adobe Firefly** - AI icon generator

### Icon Generation Services:

## Quick Setup

For now, you can use SF Symbols in Xcode:

1. In Xcode, select `Assets.xcassets`
2. Select `AppIcon`
3. Drag and drop icon images for each size
4. Or use a service like https://appicon.co to generate all sizes
